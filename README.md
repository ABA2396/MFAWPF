# MFAWPF
## 基本介绍

- 本项目是一个基于WPF框架开发的用户界面，旨在提供类似于MaaPiCli的功能

## 使用说明
### 基本说明

- 解压缩包后即可运行本项目，请确保已安装 .NET 8.0 运行时环境

## 开发相关

- 内置 MFATools 可以用来裁剪图片和获取 ROI
- 目前一些地方并没有特别完善,欢迎各位大佬贡献代码
